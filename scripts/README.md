# Scripts

Utility scripts for development/maintenance.

## Files

- **dedup_cli.py**: Deduplicate CLI commands.
- **fix_a2a_and_pipeline.py**: A2A/pipeline fixes.
- **fix_annotated.py**: Annotation fixes.
- **fix_cli_docstrings.py**: CLI docstring fixes.
- **probe_endpoints.py**: Probe nexus endpoints (visible).

Run with `python scripts/probe_endpoints.py`.

## How to start

```bash
ass-ade chat                   # default session
ass-ade chat --verbose         # show all observability events
ass-ade chat --quiet           # errors and intent only
ass-ade chat --private         # redact content from log files
ass-ade chat --dir /your/repo  # point at a specific project
```

## Core principles

| Layer | What it is |
|-------|-----------|
| `a0_qk_constants/` | Stateless invariants — constants, types, enums. Zero logic. |
| `a1_at_functions/` | Pure functions — validators, parsers, formatters. |
| `a2_mo_composites/` | Stateful classes — clients, registries, observers. |
| `a3_og_features/` | Feature modules — combining composites into capabilities. |
| `a4_sy_orchestration/` | CLI entry points and top-level orchestrators. |

Composition flows upward only. Never downward. Never sideways.
